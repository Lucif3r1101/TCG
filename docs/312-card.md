# Chronicles of the RIFT: Card Set Bible

## Purpose

This document defines the canonical structure, naming, visual identity, and narrative function of all 312 cards in **Chronicles of the RIFT**.

## Set Structure

- Total factions: 6
- Cards per faction: 52
- Total cards: 312
- Slug pattern: `<faction-id>-c01` ... `<faction-id>-c52`

Per faction:
- `c01-c32`: Units
- `c33-c52`: Spells

Rarity ladder per faction:
- Common: `c01-c29`
- Rare: `c30-c43`
- Epic: `c44-c51`
- Legendary: `c52`

## Faction Visual Canon

### Riftforged Sentinel
- Palette: cyan, steel blue, white
- Visual grammar: hard-light geometry, bastion armor, shield drones
- Unit look: disciplined military silhouettes and fortified mechs
- Spell look: walls, barriers, command glyph overlays

### Void Ranger
- Palette: indigo, neon cyan, obsidian
- Visual grammar: vectors, trails, diagonal speed language
- Unit look: agile stalkers, marksmen, phase beasts
- Spell look: blink lanes, puncture beams, trap beacons

### Ember Arcanist
- Palette: ember orange, magenta, ash black, teal sparks
- Visual grammar: runic rings, chain lattices, ignition nodes
- Unit look: arcane acolytes, ember guardians, molten familiars
- Spell look: combustion circles, combo chains, overload arcs

### Ironbound Beastmaster
- Palette: rust red, brass, moss green
- Visual grammar: claws, impact fractures, primal harnessing
- Unit look: bonded beasts, pack riders, iron-tusk brutes
- Spell look: frenzy rites, roar pulses, pack momentum surges

### Chronomancer
- Palette: gold, azure, pale violet
- Visual grammar: circles, arcs, repeated echoes, clock motifs
- Unit look: time adepts, stasis knights, chrono constructs
- Spell look: rewind sigils, delay fields, split-timeline effects

### Abyss Revenant
- Palette: deep violet, blood crimson, bone white
- Visual grammar: torn arcs, soul wisps, chain motifs
- Unit look: revenant knights, wraith hulks, soulbound hunters
- Spell look: siphon curses, resurrection marks, abyssal waves

## Rarity Identity Rules

- Common: cleaner framing, direct scene readability, low ornament density.
- Rare: stronger lighting contrast and secondary motif overlays.
- Epic: ceremonial framing and large-scale environmental drama.
- Legendary: singular iconic composition tied to each faction’s final arc moment.

## Naming Doctrine

Card names should be:
- 2 to 4 words.
- Distinct per faction voice.
- Action-first for spells.
- Title-style for units.

Examples:
- Sentinel unit: `Bastion Halberdier`
- Sentinel spell: `Aegis Resonance`
- Void unit: `Phasebound Tracker`
- Void spell: `Nullstep Ambush`

## Production Doctrine

For each faction, create in this sequence:
1. `c52` (legendary anchor)
2. `c44-c51` (epic identity set)
3. `c01-c16` (core roster)
4. remaining cards

This keeps each faction coherent before full completion.

## Final Canon Rule

Every card must communicate three things at first glance:
1. Which faction it belongs to.
2. Whether it is a unit or spell.
3. Its place in the rarity ladder.
